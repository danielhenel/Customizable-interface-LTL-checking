Upload File

On the start page, you should upload an input file either in CSV or XES format. Any other file format could not be accepted. To do so you have two possibilities:

  • Drag & drop a file from your device
  • Click on "Browse File" and choose a file.

Select features

You could select required columns from the input data you have uploaded and rename them.

Exactly one column should be selected for each required column. Otherwise the column will be not be selected for the next steps.

select LTL Rules

You can select different LTL filters and combine them with logical operators in a flexible way.

Here is a subset of the LTL filters you can apply:

  • Four eyes principle: two activities in a case cannot be executed from the same resource.
  • Attribute value different persons: indicates that an activity has been executed in a case by different resources, signaling the workload.
  • Eventually follows: finds the cases where an activity has been followed in the near/distant future by another activity.

Each filter is depending on some parameters (for example, the four eyes principle depends on two activities of the log).

The different filters can be combined in AND/OR.
